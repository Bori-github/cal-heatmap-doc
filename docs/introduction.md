Cal-Heatmap is a javascript library to create a calendar heatmap.

<hr/>

## Features

- Date locale and timezone support
- Date Navigation
- Various time granularity
- Full color customisation
- Animation
- Extension via plugins
- Typescript

See the [examples](/examples) section
